5/9（六）我會在 Claude Community Taiwan 工作坊的 panel（講者座談）上分享 AI 實戰應用與踩雷經驗

我想先蒐集大家對 Claude 或 Claude Code 最常卡住的地方 — 不管你是剛開始接觸，還是已經在用但想用得更深

我猜常見的幾個卡關點：

1. 別人說得很神，但怎麼自己做出來的東西差這麼多
2. 跟 Gemini 或是 ChatGPT 比起來差在哪
3. 大家都說要用 skill，不管自己建或是用別人的，但好像也沒有比較好
4. 不知道 Claude Cowork（在 Claude 桌面 app 裡跑任務）跟 Claude Code（在 terminal 裡寫程式）的差異
5. 要一堆權限但都不知道那是做什麼的
6. 做出來的網頁可以給別人用嗎？
