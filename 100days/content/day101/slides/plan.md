# Day 101 — Slide Plan

**Title:** Day 101 從網頁到 Mac 劉海，工具一直在找新的落腳點
**Hook:** 應用程式住的地方，這幾年一直在搬家。而且搬家的方向滿有規律的——一直往使用者注意力已經在的地方靠近。

## Visual system (locked for all slides)

- **Style:** `Dark Mode Tech` — sleek, authoritative, modern; evokes the glow of a Mac notch / Dynamic Island
- **Palette:** `Electric Dark` — charcoal `#1A1A2E`, blue `#4A90E2` + lime `#AAFF00`, white text
- **Typography:** bold sans-serif, strong size hierarchy — principle > hero element > caption
- **Corner signature:** bottom-right corner, small white monospace text `Day 101 · N/6` (where N is the slide index)
